Un repositorio dedicado a la recolección de cartas personalizadas (Custom Cards), creada por BisonWarp001.

Solo seguir los siguientes pasos: 
1. Descargar el archivo user_configs.json dentro de la carpeta config.
2. En EDOPRO, dirigirse a la carpeta "config", ingresar y pegar el archivo. Si ya han editado su propio archivo anteriormente, solo copiar la parte necesaria, para no perder sus cartas o enlaces.
3. Ingresar a EDOPRO y verificar que se descargue la información. En ocasiones requiere cerrar el juego y reiniciar.
4. Por último, en editor del Deck, buscar el apartado "Formatos Alternativos" y seleccionar la limitación "Custom".
5. Algunos Deck ya están hechos para hacer tests sin necesidad de crearlos desde cero.
